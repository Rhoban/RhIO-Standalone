# RhIO Standalone demo

To try this, you can first clone this repository:

    git clone git@github.com:Rhoban/RhIO-Standalone.git

Then, install dependencies:

    ./install-deps.sh

And build everything:

    mkdir build
    cd build
    cmake ..
    make

You can then run the main program:

    ./bin/main

And the RhIO shell:

    ./bin/rhio
